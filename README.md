# Monster Summoner

Monster Summoner is a Foundry VTT module for summoning and placing monsters through a guided interface.

## Features

- Guided summon dialog for quick monster placement
- Intended for fast GM workflows during live play
- Localized user facing strings
- Template driven UI with custom styling

## Installation

1. Open **Add-on Modules** in Foundry VTT.
2. Click **Install Module**.
3. Paste this manifest URL:
   `https://github.com/LisaNeoGermany/monster-summoner/releases/latest/download/module.json`
4. Click **Install**.

## Compatibility

- Minimum Foundry version: 10
- Verified Foundry version: 14
